About This Website

This project allows users to analyze the sentiment of product reviews using machine learning. You can enter reviews, see sentiment summaries, and view sentiment graphs.

Steps to use:

  • Login to access the dashboard.
  • Go to Analyze to input product and review details.
  • Check Home to see previous analyses.
  • View Graphs to explore insights.
